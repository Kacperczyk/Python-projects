Rock-Paper-Scissors
The player has the option to choose rock, paper, scissors, or simply enter 1) 2) 3).
The game displays both the player's and the computer's choices.
The game keeps track of the number of points scored and sums up the results.
The game asks the player if they want to play again.
The game stores the history of the player's and computer's choices, as well as the game results.
The game saves the history of choices and results to a txt file as a list.
The code responsible for saving the history and results is separated into functions, allowing for a more modular code and easier management.
